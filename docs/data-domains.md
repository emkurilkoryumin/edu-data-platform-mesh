# Домены данных онлайн-галереи

## `artwork_catalog`

Отвечает за карточки работ: название, техника, категория, изображение, статус публикации,
возрастная группа, связь с выставкой. Основной потребитель: витрина каталога и модерация.

## `young_artist_profile`

Профили юных художников: возраст, регион, наставник, дата регистрации, согласия на публикацию.
Домен нужен для персонализации, аналитики участия и Feature Store.

## `gallery_engagement`

События посетителей: просмотр работы, лайк, комментарий, добавление в избранное, вход в
виртуальный зал. Это основной домен для аналитики вовлечённости и real-time dashboard.

## `submission_moderation`

Процесс загрузки и проверки работ: SLA модерации, автоматические проверки, причины отклонения,
очередь публикации.

## `virtual_exhibition_infrastructure`

Техническая активность виртуальных выставок: залы, вместимость, активные сессии, ошибки,
время ответа, доступность.
